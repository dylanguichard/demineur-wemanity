# demineur-wemanity
Exercice du démineur en JS pour Wemanity

Bonjour ! 

Le jeu se lance en ouvrant "index.html", mais je pense que vous le saviez déjà.

Sinon j'ai remarqué en fin d'exercice que dans le vrai démineur on prends aussi en compte les cases diagonales pour comptabiliser les bombes...
Mais comme vous m'aviez dis de ne pas y passer trop de temps je n'ai pas modifier mon code sur ce point.
Le mien ne prends en compte que les cases adjacentes du coup j'ai ajouté une aide qui permet de voir les bombes (ca aide!).

Sur ce bon jeu :)
